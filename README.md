# 🎮 Arduino DIY Racing Simulator

Control racing games using a potentiometer for steering and two push buttons for gas and brake — built with an Arduino UNO and a Python bridge script.

---

Video Link :

https://youtu.be/jdcBFBj4uE4?si=Bw4w99bJvMBIjG6j


## Hardware Required

| Component | Quantity |
|---|---|
| Arduino UNO | 1 |
| 33k Potentiometer | 1 |
| Push Buttons | 2 |
| Jumper Wires | A few |

---

## Wiring

| Component | Arduino Pin |
|---|---|
| Potentiometer middle pin | A0 |
| Potentiometer left pin | 5V |
| Potentiometer right pin | GND |
| Gas Button (one leg) | Pin 8 |
| Gas Button (other leg) | GND |
| Brake Button (one leg) | Pin 7 |
| Brake Button (other leg) | GND |

---

## Controls

| Action | Input |
|---|---|
| Steer Left | Rotate pot left |
| Steer Right | Rotate pot right |
| Accelerate | Button on Pin 8 |
| Brake | Button on Pin 7 |

---

## Software Requirements

- [Arduino IDE](https://www.arduino.cc/en/software)
- [Python 3](https://www.python.org/downloads/)
- [vJoy](https://sourceforge.net/projects/vjoystick/) — virtual joystick driver
